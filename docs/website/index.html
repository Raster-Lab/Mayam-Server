Mayam

Departmental PACS — Built in Swift

Swift 6.2 DICOM 2026a DICOMweb HL7 / FHIR R4 macOS & Linux Docker v1.0.0

Features

Everything you need in a modern departmental PACS.

🔗

DICOM Networking

Full C-ECHO, C-STORE, C-FIND, C-MOVE, and C-GET SCP/SCU with TLS 1.3. 22 Storage SOP Classes and 15 Transfer Syntaxes.

🌐

DICOMweb

WADO-RS, QIDO-RS, STOW-RS, UPS-RS, and WADO-URI for standards-compliant RESTful access to imaging data.

🖼️

Image Codecs

JPEG 2000, HTJ2K, JPEG-LS, JPEG XL, and RLE via J2KSwift, JLSwift, JXLSwift, and OpenJP3D.

🖥️

Web Admin Console

Responsive web console with JWT authentication, setup wizard, and full CRUD management of nodes, storage, and settings.

🏥

HL7 & FHIR

HL7 v2.x MLLP integration for ADT/ORM/ORU messages and FHIR R4 REST endpoints for Patient, ImagingStudy, and more.

🔒

Security

IHE ATNA audit logging, DICOM anonymisation, delete protection, privacy flags, LDAP/AD, RBAC, and ACLs.

📊

Monitoring

Prometheus metrics, Grafana dashboards, health endpoint, graceful shutdown, and automated database migrations.

🐳

Docker Support

Multi-architecture container images (amd64, arm64), Docker Compose deployment, plus Homebrew and APT/RPM packages.

Quick Start

Get Mayam running in minutes.

# Pull and run the latest image
docker pull ghcr.io/raster-lab/mayam:1.0.0
docker run -d \
  -p 11112:11112 \
  -p 8080:8080 \
  -p 8081:8081 \
  ghcr.io/raster-lab/mayam:1.0.0
# Install via Homebrew (macOS)
brew tap raster-lab/mayam
brew install mayam

# Start the server
mayam-server --config /usr/local/etc/mayam/config.yaml
# Clone and build from source
git clone https://github.com/Raster-Lab/Mayam.git
cd Mayam
swift build -c release

# Run the server
.build/release/MayamServer --config Config/config.yaml
